# jQuery Interaction Demo — CSC356 Final Project

## Overview

This project is a small web application that demonstrates client-side interactivity using jQuery, along with basic PHP integration. It showcases how JavaScript, CSS, and PHP can work together to create dynamic user experiences.

## Features

- Interactive UI behavior using jQuery  
- Custom styling with CSS  
- PHP-based page rendering  
- Simple client-server interaction  

## Project Files

- `jquery.css` — styles for the application  
- `jquery.js` — jQuery-based client-side logic  
- `jquery.php` — main PHP page that loads and runs the demo  

## Requirements

- Web browser (Chrome, Firefox, etc.)  
- PHP (for running the `.php` file) or a local web server
